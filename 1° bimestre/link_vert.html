HTML

O que é?

HTML (acrônimo para a expressão inglesa HyperText Markup Language) é uma linguagem de marcação utilizada para produzir páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Sobre o curso:

Este curso ensina desde os conceitos básicos da linguagem HTML, até a criação de formulários para a Web, ideal para iniciantes, o par quem quer um guia de consulta, também possui um Glossário com todas as palavras mais utilizadas na Internet.
